# kastle2

> KASTLE 2 FX Wizard with 9 modes and DJ filter

## Controls

- **E1** — select FX mode (delay, flanger, freezer, panner, crusher, slicer, pitcher, replayer, shifter)
- **E2** — parameter 1 (mix / time / rate)
- **E3** — parameter 2 (feedback / depth / pitch)
- **K2** — hold for tap tempo
- **K3** — toggle freeze / latch

## Grid

None (parameters via encoders).

## Requirements

- norns
- engine: Kastle2 (custom SC engine)

## Install

```
;install https://github.com/jamminstein/kastle2
```
